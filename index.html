How to use

  1. Load a floor plan (image or PDF) with the green button.
  2. Calibrate:
    • Select “Two-Point Distance”, tap two known points, enter the real distance.
    • Select “Set Start Point”, tap your current location on the plan.
  3. Move/rotate using the navigation buttons:
    • “↑ Step” moves ~1 meter forward.
    • “↶ 10°” / “↷ 10°” rotate heading.
  4. Add reference points for corrections:
    • Tap “Add Reference Point”, tap location on plan, allow geolocation.
    • Use “Snap Here” in the list to jump to a reference.
  5. Use Settings to choose correction mode and toggle reference markers.
  6. Pinch to zoom, drag to pan the floor plan.
